# Usage

- change directory to `code/`
- compile: `make`
- run: `./tsp.out input_filename.txt` (outputs to `input_filename.txt.tour`)
- example: `./tsp.out data/tsp_example_1.txt` (output found at `data/tsp_example_1.txt.tour`)
